## How to install it ?

1. Install **[Stylus](https://add0n.com/stylus.html)** or similar extension which enables the browser to edit styles.
2. Go to [UserStyles](https://userstyles.world/style/14436/dev-together-2019) and install the style.
3. Enjoy!
